# Caught-in-4k-uap — Google Sheets Project

This project analyzes government-reported UFO/UAP incidents using Google Sheets. The workbook includes raw data, cleaned data, analysis tables, charts, and a dashboard.
- 
## Skills used
- data cleaning
- filtering
- find and replace
- duplicate removal
- formula building
- pivot tables
- chart creation
- dashboard design
- 
## Project structure
- `1_Raw_Data`
- `2_Cleaned_Data`
- `3_Analysis`
- `4_Visualizations`
- `5_Dashboard`
- `6_Sources`

## Sources
- AARO Official UAP Cases
- ODNI / DoD Annual UAP Report
